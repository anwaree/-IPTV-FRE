# سيرفرات IPTV المجانية

## نظرة عامة
تطبيق ويب لعرض روابط سيرفرات IPTV المجانية للمستخدمين مع عداد تنازلي قبل عرض الروابط.

## المميزات
- ✅ عرض روابط سيرفرات IPTV
- ✅ عداد تنازلي قبل عرض الروابط
- ✅ تحميل ملفات M3U لكل سيرفر
- ✅ واجهة عربية بالكامل
- ✅ تصميم بسيط وسهل الاستخدام

## البنية
- `index.html` - الصفحة الرئيسية
- `app.js` - المكون الرئيسي
- `components/CountdownButton.js` - زر العد التنازلي
- `components/ServerDisplay.js` - عرض السيرفرات
- `utils/servers.js` - قائمة السيرفرات

## كيفية الاستخدام
1. المستخدم يضغط على زر "احصل على الروابط"
2. يظهر عداد تنازلي لمدة 5 ثواني
3. تظهر قائمة السيرفرات مع أزرار التحميل
4. يمكن تحميل ملف M3U لكل سيرفر
